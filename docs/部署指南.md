# 面悟部署指南

## 部署架构

- **前端**: Vercel (免费，自动分配域名如 `mianwu.vercel.app`)
- **后端**: Railway (免费层，支持 Python)

---

## 第一步：登录 GitHub CLI

```bash
gh auth login
```

选择：
1. GitHub.com
2. HTTPS
3. 用浏览器登录

---

## 第二步：创建 GitHub 仓库并推送代码

### 前端仓库

```bash
cd /Users/ccia/Desktop/nocode

# 初始化 Git
git init
git add .
git commit -m "Initial commit: 面悟前端"

# 创建 GitHub 仓库
gh repo create mianwu-frontend --public --source=. --push
```

### 后端仓库

```bash
cd /Users/ccia/mianwu-backend

# 初始化 Git
git init
git add .
git commit -m "Initial commit: 面悟后端"

# 创建 GitHub 仓库
gh repo create mianwu-backend --public --source=. --push
```

---

## 第三步：部署后端到 Railway

1. 访问 [Railway.app](https://railway.app)
2. 点击 "Start a New Project"
3. 选择 "Deploy from GitHub repo"
4. 选择 `mianwu-backend` 仓库
5. Railway 会自动检测 Python 项目并部署

### 配置环境变量（在 Railway Dashboard 中设置）

```
DATABASE_URL=sqlite:///./data.db
OPENAI_API_KEY=你的API密钥
DEBUG=false
```

部署完成后，Railway 会分配一个域名如 `mianwu-backend.up.railway.app`

---

## 第四步：部署前端到 Vercel

1. 访问 [Vercel.com](https://vercel.com)
2. 点击 "Add New Project"
3. 选择 "Import Git Repository"
4. 选择 `mianwu-frontend` 仓库
5. 点击 "Deploy"

### 配置环境变量（在 Vercel Dashboard 中设置）

```
VITE_API_BASE_URL=https://你的后端域名（Railway分配的）
```

部署完成后，Vercel 会分配域名如 `mianwu.vercel.app`

---

## 第五步：验证部署

1. 访问前端域名，检查页面是否正常
2. 测试登录、录音等功能
3. 检查 API 是否能正常调用后端

---

## 注意事项

1. **语音识别模型**: Railway 免费层内存有限，FunASR 模型可能无法加载。建议：
   - 使用外部语音识别 API（如阿里云语音服务）
   - 或升级 Railway 付费计划

2. **数据库**: SQLite 在 Railway 上会丢失数据（容器重启）。建议：
   - 使用 Railway 提供的 PostgreSQL
   - 或使用外部数据库服务

3. **API 密钥**: 确保 OPENAI_API_KEY 正确配置

---
